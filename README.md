# Mistral Vibe (Experimental Fork)

> ⚠️ **This is an experimental playground fork with custom modifications.**

## Installation

```bash
# Install directly from git
uv pip install git+https://github.com/ohmyarthur/mistral-vibe.git

# Or clone and install
git clone https://github.com/ohmyarthur/mistral-vibe.git
cd mistral-vibe
uv sync
```

## Usage

```bash
export MISTRAL_API_KEY=your_key_here
uv run vibe
```
